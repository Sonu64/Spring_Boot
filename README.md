### 🎯 Objective
To build a deep understanding of the **Spring Boot** ecosystem, moving from core fundamentals to production-ready APIs.

### 🧩 Core Topics
* **Dependency Injection & IoC:** Managing the "Bean" lifecycle.
* **Spring MVC:** Building RESTful endpoints.
* **Data Access:** Integration with Hibernate and JPA.
* **Security:** Implementing robust authentication and authorization.

### 📝 Notepad Summary
* **Startup:** `SpringApplication.run()`
* **Metadata:** Annotations (`@Component`, `@Autowired`) drive the logic.
* **Config:** Managed via `application.properties` or `.yml`.
* **Scope:** Singletons are the default for efficient resource sharing.
